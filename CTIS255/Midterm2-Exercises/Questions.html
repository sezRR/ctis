Question #1

Take two numbers and find the minimum and maximum one. Display the result in a blue paragraph if min and max are different, otherwise display in green color. The results are centered in the page. No need to check if the user input is a number or not.


Question #2

Take three numbers, check if they are number or not. If all three input are numbers, then show their values and average in a paragraph. Otherwise, display an error message (At least one of your numbers is not correct) in red color. Paragraphs are centered.


Question #3

Get two numbers (lower and upper limit) from the user. The numbers must be within 0 and 20 inclusive. Display numbers in an unordered list starting from lower limit to upper limit. Display the sum of all those numbers in a paragraph. If the first number (lower) is not lower than the second number (upper) or if the numbers are not within the limit [0,20], then show an error message.


Question #4

Generate 10 random numbers between 10 and 30 inclusive, [10,30]. Display each number side by side whose font size is equal to that number in px.


Question #5

Generate 10 random numbers between 20 and 80 inclusive, [20,80]. First display their average in a bold text. Then, display each number in a separate paragraph. If the number is greater than average, display in blue color and bold way.

Question #14

Write a JavaScript program that imports q14_db.js file to access three object arrays and builds a report called “Attendance Report”. The report is based on the contents of those arrays. You will analyze and understand the structure of those arrays, and find the relations among them. The report basically displays the attendances of all students in the courses. For a student, it displays her name, surname, city, cgpa and attendance percentage. City will be a select box. It is not static, and it is built from the cities in the “students” array. The cities in the list are ordered alphabetically, and there is no duplication, and appropriate city is selected for each student. Attendance is a bar chart. Bar container’s width is 100px, its height is 15px, background color is #DDD, and border color is #AAA. If the attendance is greater than 60, the bar in the bar container will be green otherwise, it will be red. The bar chart’s width is given in “attendances” array for each student. Each course is a separate table with a width of 500px. It has a collapsed borders, and its border color is #AAA. It uses a strip pattern (alternating background colors, #F0F0F0 and #FFF) for student rows. The first row contains course code and name. It is bold and centered, and its background color is #CCC. The second row shows the titles of each column. They are bold and italic, and they have a bottom border which is “4px double black”. Each cell has a 5px padding.
